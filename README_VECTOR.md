# Redmine Support Tool - Redmine操作支援ツール

**「毎日のRedmine入力、もっと楽に。」**

Redmine Support Tool は、日々の作業時間記録（工数入力）を劇的に効率化するデスクトップツールです。
定例作業の自動化や、カレンダー形式での視覚的な管理により、入力の手間とストレスを最小限に抑えます。

## ✨ 主な機能

- **📅 カレンダービュー**: 月ごとの作業時間をカレンダーで可視化。登録状況が一目で分かります。
- **⚡ 一括登録**: 毎日の朝会や定例会議など、繰り返し発生するタスクをワンクリックで月全体に登録。
- **📝 テンプレート**: よく使うチケットや作業内容をテンプレートとして保存し、呼び出し可能。
- **🎨 モダンUI**: シンプルで洗練されたデザイン。直感的に操作できます。
- **🔧 安心設計**: 通信ログ機能により、どのようなデータが送信されているかを確認可能。

## 📦 インストール方法

本パッケージは **完全セットアップ済み（Standalone版）** です。

1. ダウンロードした ZIP ファイルを展開します。
2. フォルダ内の `redmineSupTool.exe` をダブルクリックして実行します。

> **Note**: .NET ランタイムがアプリに内蔵されているため、**面倒なランタイムのインストールは不要**です。
> 展開してすぐに使い始めることができます。

## 🚀 はじめかた

1. アプリを起動し、設定画面を開きます。
2. お使いの Redmine の URL と APIキーを入力します。
   - *APIキーは Redmine の「個人設定」画面右側から取得できます。*
3. 「接続テスト」をクリックして接続を確認します。
4. メイン画面に戻り、「テンプレート追加」から日々のタスクを登録してみましょう！
