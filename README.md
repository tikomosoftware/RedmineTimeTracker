# Redmine Time Tracker

**[English](README.en.md)** | 日本語

---

**Redmineの作業時間入力、もっと楽にしませんか？**

Redmine Time Tracker は、毎日の作業時間記録を効率化・自動化するためのデスクトップアプリケーションです。
「チケットIDを探して、時間を計算して、入力して…」という面倒な繰り返し作業からあなたを解放します。

## 💭 開発経緯

以前は、Pythonスクリプトを使ってコマンドラインから時間入力を自動化していました。
しかし、実際に使ってみると、以下のような課題がありました：

- ❌ 登録状況が一目で分からない
- ❌ どの日に登録したか、していないかの確認が面倒
- ❌ コマンドラインは直感的ではない

そこで、**カレンダー表示**と**視覚的な操作**ができるGUIアプリケーションを開発しました。

**結果**:
- ✅ 月全体の登録状況が一目で分かる
- ✅ マウスで簡単に操作できる
- ✅ 登録漏れをすぐに発見できる

コマンドラインツールからGUIアプリへの進化により、作業時間入力がさらに快適になりました。

## ✨ 特徴

### 📅 月次カレンダービュー
1ヶ月の作業記録をカレンダー形式で一目で把握。
「どの日に」「どのチケットで」「何時間」作業したかが視覚的に分かります。
登録漏れのチェックも簡単です。

### ⚡ テンプレート一括登録
毎日・毎週発生する定例タスクはテンプレート化。
「朝会」「週次ミーティング」などの決まった作業は、月全体に対してワンクリックで一括登録できます。
もう同じ入力を毎日繰り返す必要はありません。

### 🎨 モダンで使いやすいUI
Windows の最新トレンドを取り入れた、シンプルで美しいデザイン。
直感的な操作感で、マニュアルなしですぐに使えます。

### 🔍 透明性のある通信ログ
アプリが Redmine と行っている通信（APIリクエスト）を確認できるログビューアを搭載。
「何が送信されたか」が分かるので、安心して利用できます。

## 📦 パッケージについて

本ソフトウェアは用途に合わせて2つのパッケージを用意しています。通常は **Standalone** 版をおすすめします。

### 1. Standalone (推奨)
- **ファイル名**: `RedmineSupTool-vX.X.X-standalone.zip`
- **特徴**: .NET ランタイムを同梱しています。
- **メリット**: **事前のインストール作業やセットアップが一切不要**です。解凍してすぐ実行できます。

### 2. Framework-dependent (軽量版)
- **ファイル名**: `RedmineSupTool-vX.X.X-framework-dependent.zip`
- **特徴**: アプリケーション本体のみのパッケージです。
- **条件**: 別途 [.NET 9.0 Runtime](https://dotnet.microsoft.com/download/dotnet/9.0) のインストールが必要です。
- **メリット**: ファイルサイズが非常に軽量です。

## � 動作要件

- **Redmine**: バージョン 3.0 以降 (REST API が有効であること)
- **OS**: Windows 10 以降

## �🚀 使い方

1. `redmineSupTool.exe` を起動します。
2. 初回起動時に Redmine の **URL** と **API Key** を設定します。
3. 接続テストを行い、問題なければ準備完了です。
4. テンプレートを作成したり、カレンダーから作業時間を登録してください。
